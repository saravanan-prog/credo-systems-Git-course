# Add file
```
Step 1 : Create a New File
Step 2 : Click Add file → Create new file.
Step 3 : Enter the file name (for example, README.md or index.html).
Step 4 : Enter the file content.
Step 5 : Scroll down to Commit changes.
Step 6 : Click Commit changes.
```

# Upload a File
```
Step 1 : Click Add file → Upload files.
Step 2 : Drag and drop the file, or click choose your files.
Step 3 : Wait for the upload to finish.
Step 4 :  Enter a commit message.
Step 5 : Click Commit changes.
```

# Edit  a File
```
Step 1: Open your repository on GitHub.
Step 2: Click the file you want to edit 
    (for example, README.md or index.html).
    
Step 3: Click the ✏️ Edit this file (pencil icon) at the top-right of the file view.
Step 4: Make your changes in the editor.
Step 5: Scroll down to the Commit changes section.
Step 6: Enter a commit message, such as:
Step 7: Updated README
Step 8: Fixed HTML content
Step 9: Click Commit changes.

Your file is now updated in the repository.

```
