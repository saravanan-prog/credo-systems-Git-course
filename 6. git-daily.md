#### 1. Clone the Repository

```
git clone https://github.com/company/project.git
```

#### 2. Get the Latest Changes

```
git pull origin main

```

### 3. List all branch

```
    git fetch --all
    git branch -a

```

### 4. Switch the branch

```
git checkout  feature/login-page

```

#### 5. Create a New Branch
```
git checkout -b feature/login-page
```
