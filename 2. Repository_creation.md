# Repository Creation

Step 1. 
```
     Sign in to your GitHub account at GitHub.
```
Step 2. 
```
        Click the + icon in the top-right corner.
```
Step 3. 
```
        Select New repository.
```
Step 4.

```
        Enter a Repository name (for example: my-project).
```
Step 5. 
```
        (Optional) Add a Description.
```
Step 6.
```
 Choose:

        Public      – anyone can view the repository.
        Private     – only authorized users can view it.
```

Step 7.
```
 Select Add a README file (recommended).
```
Step 8. 
```
(Optional) Choose a .gitignore template and a License.
Click Create repository.
```
