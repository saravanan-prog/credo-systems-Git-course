# Git Hub Signin

Step 1 
```
    Go to GitHub Sign Up
```
Step 2  
```
    Click Continue with Google.
```
 Step 3
```
    Select your Gmail/Google account.
```
Step 4
```
    Sign in to Google if prompted.
```
Step 5
```
    Grant GitHub permission to access your Google account.
```
Step 6
```
    Choose a GitHub `username` if asked.
```
Step 7
```
    Complete any verification steps.
```
Step 8
```
    Finish the account setup.
```
