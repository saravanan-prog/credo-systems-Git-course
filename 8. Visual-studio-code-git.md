# Clone a Git Repository in VS Code
```
Step 1 : Open VS Code.
Step 2 : Press Ctrl + Shift + P to open the Command Palette.
Step 3 : Type Git: Clone and select it.
Step 4 : Paste the repository URL
```

# Commit Changes in VS Code
Step 1 :
```
 Open your project in VS Code.
```
Step 2 :
```
 Click the Source Control icon (or press Ctrl+Shift+G).
```
Step 3 :
```
 Review the changed files under Changes.
```
Step 4 : 
```
Click a file to view the diff.
```
Step 5 : 
```
Stage files:
            Click the + icon next to each file, or
            Click + next to Changes to stage all files.
```

Step 6 : 
```
Enter a commit message in the message box at the top.

            Example : Fix login validation issue
```
Step 7 :
```
 Click Commit (✓ icon) or press Ctrl+Enter.
```
Step 8 :
```
 Click Sync Changes or Push from the Source Control menu.
```

# Create a Git Branch in VS Code

```
Step 1 : Open your project in VS Code.
Look at the bottom-left corner where the current branch name is displayed (e.g., main).

Step 2 : Click the branch name.

Step 3 : Select Create New Branch.

Step 4 : Enter a branch name, for example : feature/login-page
Press Enter.

Step 5 : VS Code will create the branch and switch to it automatically.
```
